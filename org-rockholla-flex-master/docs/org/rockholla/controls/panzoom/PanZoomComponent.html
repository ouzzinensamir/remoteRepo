Packageorg.rockholla.controls.panzoom
Classpublic class PanZoomComponent
InheritancePanZoomComponent mx.containers.Canvas

Language Version : ActionScript 3.0

The PanZoomComponent is a Flex 3 and 4 compatible control, capable of laying out flex components within a container that can be panned and zoomed via dragging, dropping, and mouse wheel operation.
